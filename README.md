# NCTO — optically written zigzag order in Na₂Co₂TeO₆

Source of truth for the paper *"One-defect picture for long-lived optically
written zigzag order in Na₂Co₂TeO₆"* and its driven-dynamics campaign.

This repo holds **source only** (paper, scripts, cluster job package, small
reference states, final figures). The heavy simulation outputs (HDF5
trajectories, per-run directories) are regenerated by the campaign and are **not**
tracked — see `NCTO_project/RUN_PLAN.md` for what to run.

## Layout

```
NCTO_project_paper/
  phonon_engineering_ncto.tex        # the paper
  phonon_engineering_nctoNotes.bib
NCTO_project/
  RUN_PLAN.md                        # <-- remaining simulations to run
  scripts/                           # campaign drivers, plotters, np_campaign helpers
  cluster_campaign/                  # SLURM package (build + run on a cluster)
  tuned_kruger_campaign/             # small reference states + the final figures
```

## Physics summary (what changed and why)

- The E1 magnetoelastic coupling is the **quadratic (nonlinear-phononic)**
  exchange striction actually implemented in the solver, on a single
  zone-center mode **Q**=(Qₓ,Q_y):
  `δX_γ(Q)=λ_{X,0}|Q|² + λ_{X,2}[(Qₓ²−Q_y²)cos2θ_γ + 2QₓQ_y sin2θ_γ]`.
- Uniform (signed) Grüneisen coupling `λ_{X,2}=(X/K)·λ_{K,2}` with `λ_{K,2}=0.02`
  (the earlier `|X|/|K|` choice had the wrong sign on the J and Γ channels).
- **J7 ring exchange is phonon-tunable**: a hexagon is C₆-symmetric, so the ring
  couples isotropically, `δJ₇(Q)=λ_{J7,0}|Q|²` (breathing only). Sensible
  strength `λ_{J7,0}=(J7/K)λ_{K,2}=1.0×10⁻³` (matches the bilinear channels'
  fractional modulation).

## Running the campaign

The scripts depend on the ClassicalSpin_Cpp solver (`build/spin_solver`) and its
`util/` helpers; they expect to run **inside a ClassicalSpin_Cpp checkout**.
On a cluster, follow `NCTO_project/cluster_campaign/README.md`:

1. `git clone git@github.com:ze-bang/ClassicalSpin_Cpp.git` and build the solver.
2. `git clone git@github.com:ze-bang/NCTO.git` and copy its `NCTO_project/` and
   `NCTO_project_paper/` on top of the ClassicalSpin_Cpp checkout.
3. `sbatch NCTO_project/cluster_campaign/run_campaign.sbatch`.

See `NCTO_project/RUN_PLAN.md` for exactly which runs remain and their commands.
